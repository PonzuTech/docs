## Per-Project Contracts

Each token deployment creates unique contract addresses for:
- Token (ERC-20)
- Presale
- Launcher
- Distributor
- Farm
- PonzuBottle (ERC-721)
- LiquidityCard (ERC-721)
- Project
- Operator

These addresses are returned by the `craftPonzu` transaction and emitted in the `PonzuCrafted` event. See [PonzuRecipe](/protocol/recipe) for parsing details.

## PonzuSwap Pair Addresses

Trading pairs are created at launch (not at deployment). After `triggerLaunch()` is called, query the pair:

```typescript
const pairAddress = await publicClient.readContract({
  address: '0x1DCA548D67938E6162f0756985cC3e539Aae30C2', // PonzuSwap Factory
  abi: parseAbi(['function getPair(address, address) view returns (address)']),
  functionName: 'getPair',
  args: [tokenAddress, WETH],
})
```
