## What You Deploy

One transaction through PonzuRecipe. Eight contracts, all immutable, all interconnected:

| Contract | What It Does |
|----------|-------------|
| **Token** | Your ERC-20. 1,000,000 fixed supply. No mint function exists |
| **Presale** | Sells 690,000 tokens (69%) at linear pricing |
| **Launcher** | Creates PonzuSwap liquidity when presale fills |
| **Distributor** | Routes swap fees and forfeited tokens to stakeholders |
| **Farm** | LP staking rewards for your community |
| **Ponzu Bottle** | ERC-721 NFT representing presale positions |
| **Liquidity Card** | ERC-721 NFT representing farm positions |
| **Project** | Registry and coordinator contract |

**Cost:** 0.005 ETH creation fee + optional dev buy amount (your own presale purchase).

**Verification:** All eight contracts deploy from the PonzuRecipe factory. No proxy patterns. No upgrade functions. No admin keys. Read the source on Etherscan. Search for `selfdestruct`, `delegatecall`, `upgradeTo`. None exist.

## Configuration

Set at deployment. Fixed permanently after.

| Parameter | Options | Notes |
|-----------|---------|-------|
| Token Name | Any string | Display name |
| Token Symbol | Any string | Ticker (e.g., MYTKN) |
| Vesting Duration | 10 days (Yellow Bottle) or 10 weeks (Green Bottle) | See [diamond-hand vesting](/raise/diamond-hand-vesting) |
| Target Raise | Sets the linear price curve | Minimum 3 ETH on mainnet. `endPrice = targetRaise / 690,000` |
| Metadata | IPFS, Arweave, or HTTPS URI | Image, description, social links |
| Dev Buy | Optional ETH amount | Your own presale purchase. Same vesting as everyone else |

## Your Revenue

Every ETH you earn, when you earn it, where it comes from. Worked example: **50 ETH target raise**.

### During Presale
- **1% creator fee** on every purchase
- Arrives in your wallet immediately
- On 50 ETH raise: ~0.52 ETH total

### At Launch
- **9% of the presale pool** goes directly to your wallet
- The presale pool is 96% of all ETH contributed (after the 4% fee)
- On 50 ETH pool: **4.5 ETH** immediate runway

### Post-Launch (Ongoing)
- **15% of Distributor token allocation** from forfeited vesting tokens
- **15% of Distributor ETH allocation** from the 1% swap fee on every PonzuSwap trade

### Total from a 50 ETH Raise

| Source | Amount | When |
|--------|--------|------|
| Creator fee (1%) | ~0.52 ETH | During presale, per purchase |
| Treasury (9%) | 4.5 ETH | At launch, one transaction |
| Distributor (15% of tokens) | Variable, ongoing | Post-launch, from early claimers |
| Distributor (15% of ETH fees) | Variable, ongoing | Post-launch, from swap volume |
| **Guaranteed minimum** | **~5.02 ETH** | Before any trading occurs |

Post-launch revenue depends on trading volume and how many holders claim early. Active trading generates continuous ETH fees. 60% of holders claim on Day 1, 15% of their forfeited tokens go to your treasury.

## The Founder Workflow

<Steps>
  <Step title="Deploy">
    Configure your token and deploy through ponzu.app or the SDK. One transaction. All eight contracts are live and immutable.
  </Step>
  <Step title="Build in Public">
    The presale runs in the background. Ship features while the presale fills. Your 1% creator fee arrives with every purchase. No pitch decks. No term sheet negotiations.
  </Step>
  <Step title="Launch Happens Automatically">
    When 690,000 tokens sell, anyone calls `triggerLaunch()`. PonzuSwap liquidity is created permanently. Your 9% ETH arrives in one transaction. Diamond-hand vesting begins. Once your project has pumped and dumped, it's dead. Ponzu doesn't let that happen.
  </Step>
  <Step title="Earn While Building">
    15% of swap fees and 15% of forfeited vesting tokens flow to your treasury. Your community farms LP rewards. You build product. The smart contracts handle the economics.
  </Step>
</Steps>

## If the Presale Stalls

No time limit. You keep all creator fees earned so far. Buyers can refund (90% back). You can't cancel. Keep building. A presale that fills because the product proved itself is worth more than a sellout on hype.

## SDK

Deploy through ponzu.app or programmatically:

```typescript
import { deploy } from '@ponzu_app/sdk'
import { parseEther } from 'viem'

const result = await deploy(
  {
    owner:          account.address,    // Your wallet, receives treasury + creator fees
    tokenName:      'My Token',
    tokenSymbol:    'MYTKN',
    metadata:       'ipfs://Qm...',     // Project description + socials
    imageURI:       'ipfs://Qm...',     // Token image
    targetEthRaise: parseEther('5'),    // Minimum 3 ETH on mainnet
  },
  wallet,       // viem WalletClient
  client,       // viem PublicClient
  'mainnet',    // 'mainnet' or 'sepolia'
)

// All eight contract addresses are returned:
// result.addresses.token
// result.addresses.presale
// result.addresses.launcher
// result.addresses.distributor
// result.addresses.farm
// result.addresses.ponzuBottle
// result.addresses.liquidityCard
// result.addresses.project
```

The `deploy()` function sends a single transaction to PonzuRecipe. All contracts are created atomically. If any part fails, the entire deployment reverts.

Full SDK reference: [SDK on NPM](https://www.npmjs.com/package/@ponzu_app/sdk)
